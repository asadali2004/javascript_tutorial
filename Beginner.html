2.1. Variables and Data Types

A variable is a container for storing data. In JavaScript, you declare variables using let, const, or the older var keyword.

Variable Declaration Keywords

  • let: Reassignable variable with block scope
  • const: Constant value (preferred when reassignment isn't needed)
  • var: Obsolete; function-scoped (avoid using)

Primitive Data Types

JavaScript has 7 primitive data types:

string: Text data enclosed in quotes
number: Integers and floating-point numbers
boolean: true or false values
null: Intentional absence of value
undefined: Variable declared but not assigned
bigint: Large integers beyond Number.MAX_SAFE_INTEGER
symbol: Unique identifiers 
// Variable declarations
let userName = "Alice";
console.log(userName);
userName = "Bob"; // Reassignment allowed
console.log(userName);

const userEmail = "alice@example.com";
console.log(userEmail);

// Primitive data types examples
const greeting = "Hello, world!"; // string
const pi = 3.14; // number
const isLoggedIn = true; // boolean
let selectedItem = null; // null
let favoriteColor; // undefined

2.2. Operators

Operators perform actions on values and variables. JavaScript provides several types of operators for different purposes.

Types of Operators:

Arithmetic: +, -, *, /, %, ** (addition, subtraction, multiplication, division, modulus, exponentiation)
Assignment: =, +=, -=, *=, /= (assign and combine with operation)
Comparison: ==, ===, !=, !==, >, <, >=, <= (compare values)
Logical: && (AND), || (OR), ! (NOT) 
// Arithmetic operators
let total = 10 + 5; // 15
let remainder = 17 % 5; // 2
let power = 2 ** 3; // 8

// Assignment operators
let x = 10;
x += 5; // x is now 15

// Comparison operators
const a = 5;
const b = "5";
console.log(a == b);  // true (loose equality)
console.log(a === b); // false (strict equality)
console.log(a !== b); // true (strict inequality)

// Logical operators
const isAdmin = true;
const isLoggedIn = true;
const hasAccess = isAdmin && isLoggedIn; // true

Important Note: == vs ===

Always use === (strict equality) instead of == (loose equality) to avoid unexpected type coercion behavior.

  3. Practice with assignment operators: start with a number and use +=, -=, *=, /=

2.3. Control Flow

Control flow statements allow you to control the execution of your code based on conditions or repeat code multiple times.

Conditional Statements

Use if, else if, and else to execute different code blocks based on conditions.

// If-else statements
const userAge = 18;
if (userAge >= 18) {
  console.log("You are an adult.");
} else if (userAge >= 13) {
  console.log("You are a teenager.");
} else {
  console.log("You are a child.");
}

// Switch statement
const dayOfWeek = "Monday";
switch (dayOfWeek) {
  case "Monday":
    console.log("Start of the week.");
    break;
  case "Friday":
    console.log("End of the week!");
    break;
  default:
    console.log("Regular day.");
}

Loops

Loops allow you to repeat code execution multiple times.

// For loop - when you know how many iterations
for (let i = 0; i < 5; i++) {
  console.log(`For loop count: ${i}`);
}

// While loop - when condition-based iteration
let count = 0;
while (count < 3) {
  console.log(`While loop count: ${count}`);
  count++;
}

// Do-while loop - executes at least once
let num = 0;
do {
  console.log(`Do-while: ${num}`);
  num++;
} while (num < 3);

Loop Control

  • break: Exit the loop immediately
  • continue: Skip the current iteration and continue with the next

2.4. Functions

Functions are reusable blocks of code that perform specific tasks. They help organize your code and avoid repetition.

Function Benefits:

Reusability: Write once, use multiple times
Organization: Break complex problems into smaller pieces
Maintainability: Easier to update and debug code
Scope: Create isolated environments for variables 
// Function with parameters and return value
function greetUser(name) {
  return `Hello, ${name}!`;
}

const message = greetUser("Charlie");
console.log(message); // "Hello, Charlie!"

// Function without return value
function sayHello() {
  console.log("Hello there!");
}
sayHello(); // Calls the function

// Function with multiple parameters
function calculateArea(width, height) {
  const area = width * height;
  return area;
}

const roomArea = calculateArea(12, 10);
console.log(`Room area: ${roomArea} sq ft`);

Function Scope

Variables declared inside a function are only accessible within that function. This is called function scope or local scope.

function example() {
  let localVar = "I'm local";
  console.log(localVar); // Works fine
}
// console.log(localVar); // Error: localVar is not defined
